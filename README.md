# 📊 Financial Statement Dashboard (Guided Project)

## 📌 Overview

Project ini merupakan dashboard analisis laporan keuangan menggunakan **Microsoft Power BI** yang dibuat sebagai bagian dari proses pembelajaran melalui tutorial resmi dari [Microsoft Power BI](https://powerbi.microsoft.com/?utm_source=chatgpt.com).

Walaupun project ini bersifat guided/tutorial-based, project ini membantu memahami:

* data visualization,
* dashboard design,
* data transformation,
* dan dasar analisis bisnis menggunakan Power BI.

---

## 🎯 Objectives

Tujuan dari project ini adalah untuk mempelajari:

* Pembuatan dashboard interaktif di Power BI
* Penggunaan visualisasi data untuk financial analysis
* Data filtering menggunakan slicer
* Penyajian insight bisnis secara visual
* Dasar penggunaan Power Query dan DAX

---

## 📈 Dashboard Features

Beberapa fitur yang digunakan dalam dashboard:

* 📊 Sales & Profit Analysis
* 🌍 Profit Distribution by Country
* 📅 Year Filter (Slicer)
* 📦 Product Performance Comparison
* 📉 Financial Trend Visualization
